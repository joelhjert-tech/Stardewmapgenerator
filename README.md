# Stardew Map Generator

This repository currently focuses only on mine/dungeon generation.
Other themes are intentionally out of scope until mine/dungeon generation is stable.

The active pipeline uses Stardew-style mine/dungeon structural generation with approved blocks, visual canon templates, Joel-authored runs, marker fallback, validators, and prototype-only outputs.

This repository does not include Stardew Valley game assets, unpacked basegame maps, Moonvillage source maps, third-party mod art, rendered asset previews, or production map exports.
